# Areesha Dar — Computer Engineering Portfolio

A modern, responsive personal portfolio website showcasing my skills, projects, experience and frontend development journey.

The website features a blueprint-inspired design with a focus on clean UI, accessibility, responsive layouts and modern CSS techniques.


## Features

- Responsive design for mobile, tablet, and desktop
- Blueprint-inspired engineering theme
- Semantic HTML5 structure with accessibility features
- 12-column CSS Grid layout with Flexbox fallback
- Fluid typography and spacing using CSS `clamp()`
- CSS variables for consistent design system
- Smooth scrolling navigation
- Animated hero section and interactive elements
- Skills and project showcase sections
- Contact section with social links
- Download CV functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- CSS Grid
- Flexbox
- Responsive Web Design
- CSS Animations
